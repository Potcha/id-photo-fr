Guide conformité (rappel)
  • Taille photo : 35×45 mm (413×531 px à 300 DPI).
  • Hauteur du visage (menton → sommet crâne) : 32–36 mm (cible 34 mm).
  • Tête droite : yeux horizontaux ±2° (outil « Aligner les yeux »).
  • Fond uni clair (gris/bleu clair), sans ombre ni motif. Le blanc peut être refusé.
  • Visage net, expression neutre, bouche fermée, yeux ouverts.
  • Pas de filtre, lunettes teintées, couvre‑chef (sauf cas autorisés).
